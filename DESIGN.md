# 共赢挑战·同心聚力 — 游戏网站设计规范

## 一、项目概览

**活动名称**：共赢挑战·同心聚力  
**任务主题**：凝心共奋进，同心赢未来  
**任务地点**：鼎湖山蝴蝶谷  
**参与规模**：260人，10组，每组26人

## 三、信息架构

```
首页
├── 活动介绍 / 规则说明
├── 品牌组队选择（10选1，容量26/组）
└── 任务区分配（七星岩10人 / 鼎湖山16人）

主线任务 — 共赢挑战赛
├── 第一关：山水接力运臻品
├── 第二关：同心搭筑品牌塔
└── 第三关：环湖聚力拔河赛

积分榜 / 我的团队
```

---

## 四、核心交互流程

1. **组队流程**：自愿选组 → 满员提示 → 确认加入 → 生成专属团队标识
2. **任务分配**：小组内部选择七星岩(10)或鼎湖山(16)任务区
3. **游戏流程**：依次闯关 → 实时计分 → 违规扣分提示 → 最终排名

---

## 五、技术实现要点

- 响应式设计，支持PC/平板
- 分组状态实时展示（已加入人数/总容量）
- 三大关卡需支持多人协同的界面框架
- 积分榜实时更新
