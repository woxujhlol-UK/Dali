# 共赢挑战·同心聚力 — 达利同心共赢挑战赛

活动主题：**凝心共奋进，同心赢未来**  
任务地点：鼎湖山蝴蝶谷

## 项目说明

本网站为「共赢挑战·同心聚力」团建活动的线上互动平台，包含组队、任务分配、闯关游戏与积分榜等功能。

## 文件结构

```
网站A1/
├── index.html      # 主页面（活动介绍、组队、闯关、积分榜、规则）
├── game1.html      # 第一关：山水接力运臻品
├── game2.html      # 第二关：同心搭筑品牌塔
├── game3.html      # 第三关：环湖聚力拔河赛
├── styles.css      # 全局样式
├── app.js          # 主页面交互逻辑
├── DESIGN.md       # 设计规范文档
└── README.md       # 本说明
```

## 使用方法

直接用浏览器打开 `index.html` 即可访问。本地开发可配合任意静态服务器（如 `npx serve .`）运行。

## 功能概览

- **品牌组队**：10 组达利旗下品牌，每组 26 人，自愿选择，满员不可加入
- **任务区**：七星岩 10 人 / 鼎湖山 16 人，组内自行分配
- **三大关卡**：接力运臻品、搭筑品牌塔、环湖拔河赛
- **积分榜**：三关总分排名（当前为模拟数据）

## 技术栈

- HTML5 / CSS3 / JavaScript
- 响应式布局，支持 PC 与移动端
- 本地存储保存组队信息
